Данная директория содержит все заголовочные файлы и библиотеки DirectX 9.0.
Версия DirectX SDK: [Oct 2004, 9.02.3900].

Это самая последняя версия SDK со статической линковкой d3dx9.lib (не требующая d3dx9_xx.dll), и при
этом содержащая библиотеку d3dx8.lib (т.е. поддерживающая разработку под DirectX 8, только для x86).

В версии SDK [Dec 2004, 9.03.91] библиотека d3dx9.lib также имеет статическую линковку, однако среди
.lib файлов этой версии отсутствует d3dx8.lib (хотя d3d8.lib и DxErr8.lib присутствуют).

***

В директории doc лежат .chm/.chi файлы документации из DirectX SDK:
  directx8_c.* - документация для C/C++, DirectX 8.1 SDK [Oct 2001]
  directx9_c.* - документация для C/C++, DirectX 9.0 SDK [Oct 2004, 9.02.3900]
